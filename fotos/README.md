# Fotos del diario

Coloca aqui las fotos que quieras mostrar publicamente en la pagina y luego agregalas a `fotos.json`.

Ejemplo:

```json
[
  {
    "src": "vanesa-01.jpg",
    "title": "Nuestro primer cafe",
    "date": "2026-06-30",
    "note": "Una tarde tranquila que quiero recordar."
  }
]
```

El campo `src` debe coincidir exactamente con el nombre del archivo dentro de esta carpeta.

Estas fotos aparecen en la seccion **Diario** de la pagina. Si agregas una foto desde el formulario de la pagina, tambien se vera ahi, pero solo en el navegador donde la guardaste.
