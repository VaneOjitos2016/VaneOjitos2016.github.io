# Pagina para Vanesa

Repositorio estatico listo para GitHub Pages.

## Como publicarlo

1. Sube todos estos archivos a un repositorio de GitHub.
2. En GitHub, entra a **Settings > Pages**.
3. En **Build and deployment**, elige **Deploy from a branch**.
4. Selecciona la rama `main` y la carpeta `/root`.
5. Guarda los cambios y espera el enlace de GitHub Pages.

## Personalizacion rapida

- Los botones de WhatsApp ya apuntan a `51917726087`.
- Para cambiar la fecha del cronometro, edita `metAt` en `DEFAULT_SETTINGS`.
- Para agregar una foto rapido desde la pagina, entra al bloque **Diario**, llena el recuerdo, elige **Foto local** y pulsa **Guardar recuerdo**. Se vera al lado derecho en el diario, pero queda guardada solo en ese navegador.
- Para que la foto sea publica en GitHub Pages y Vanesa tambien la vea desde otro celular, copia la imagen a `fotos/` y registrala en `fotos/fotos.json`.
- Los planes realizados se guardan en el navegador con `localStorage`.
- Abajo de la pagina hay mini detalles: puzzle, ruleta, cupones, dados de cita, frasquito de razones, bingo, contador de sonrisas, codigo secreto y mini ritual.

Formato de numero recomendado para WhatsApp: codigo de pais + numero, sin `+`, espacios ni guiones. Ejemplo Peru: `51999999999`.
